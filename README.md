## ✨ Key Features

* 🔐 **User Authentication** – Secure login & session handling
* 📊 **Real-time Attendance Tracking** – Mark and monitor daily attendance
* 🧾 **OCR-based Timetable Creation** – Automatically generate schedules by scanning timetable images
* 🎵 **Interactive Attendance Marking Page** – Assign different songs for attendance types (*Present, Absent, Cancel, Medical*)
* 📳 **Haptic Feedback Integration** – Tactile feedback on key interactions for enhanced UX
* 📚 **Subject-detail Page** – View subject details with attendance stats, and historical records
* 🗓️ **Event Creation via Voice Commands** – Create reminders and events using voice input
* ⏰ **Smart Reminders** – Get notified for classes, attendance, and custom events
* 📱 **Cross-Platform Support** – Works on both Android & iOS
* 🌐 **Offline Handling** – Graceful behavior in low/unstable network

## ⚙️ Local Setup & Installation

### 🔧 Prerequisites

* Node.js (v16 or higher)
* Expo CLI

  ```bash
  npm install -g expo-cli
  ```
* Expo Go app (Android/iOS) *or* Emulator

---

### 🚀 Installation Steps

1. **Clone the repository**

   ```bash
   git clone https://github.com/KGP-Presence/Attendane-Tracker-Mobile-App.git
   cd Attendane-Tracker-Mobile-App
   ```

2. **Install dependencies**

   ```bash
   npm install
   # or
   yarn install
   ```

3. **Setup environment variables**

   Create a `.env` file in the root directory:

   ```env
   EXPO_PUBLIC_API_URL=http://your-backend-url.com/api
   ```

4. **Start the development server**

   ```bash
   npx expo start
   ```

5. **Run the app**

   * 📱 Scan QR using **Expo Go**
   * 💻 Or run on emulator
